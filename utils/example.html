This is a demonstration of the downloadPdf utility.

The content in this terminal window can be converted to PDF.

Features demonstrated:

• Static method usage

• Class-based instantiation

• Custom configuration options

• Error handling

Date:

Time:

Ready to download PDF...

Usage Examples:

1. Basic Usage (Static Method)

await DownloadPdfUtil.download({ filename: 'basic-demo.pdf' });

2. Class-based Usage

const pdfUtil = new DownloadPdfUtil({ selector: '.terminal-window', filename: 'class-demo.pdf', scale: 3 }); await pdfUtil.downloadPdf();

3. Backward Compatibility

await downloadPdf({ filename: 'compatibility-demo.pdf' });

Configuration Options:

  • selector: '.terminal-window' (CSS selector)
  • filename: 'document.pdf' (output filename)
  • scale: 2 (image quality multiplier)
  • delay: 100 (rendering delay in ms)
  • orientation: 'portrait' or 'landscape'
